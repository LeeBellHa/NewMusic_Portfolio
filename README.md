# SIRO – Artist Portfolio Website

뮤지션을 위한 현대적이고 세련된 포트폴리오 웹사이트입니다. **SIRO** 밴드의 공식 웹사이트로, 반응형 디자인과 부드러운 사용자 경험을 제공합니다.

## 🎵 소개

**SIRO**는 인간 내면의 모순과 감정을 직면하는 인디 록 밴드입니다.
- 감정의 양면성을 숨기지 않고 사운드 위에 옮겨 담습니다
- 모든 멤버가 작사·작곡·편곡에 참여합니다
- 한 곡 안에서도 감정의 고조와 폭발, 잔잔한 여운을 드라마처럼 구성합니다

---

## ✨ 주요 기능

### 📱 반응형 디자인
- 모든 기기(데스크톱, 태블릿, 모바일)에서 완벽하게 작동합니다
- 모바일 친화적인 네비게이션 메뉴

### 🎨 현대적인 UI/UX
- 밝고 따뜻한 톤의 디자인 시스템
- 부드러운 애니메이션과 트랜지션
- 제목 폰트: Pretendard 웹폰트로 한글 최적화

### 🎭 주요 섹션
1. **Hero Section** - 밴드 소개 및 프로필 사진
2. **Profile** - 밴드 소개, 멤버 정보 및 바이오그래피
3. **Discography** - 최신 앨범 및 싱글
4. **Gallery** - 밴드 활동 사진 및 공연 기록
5. **Links** - 스포티파이, 유튜브, 인스타그램, TikTok 연결
6. **Contact** - 예약 및 문의 폼

### 🔧 기술 구현
- **부드러운 스크롤**: HTML 네이티브 `scroll-behavior: smooth`
- **Sticky Header**: 스크롤 시 네비게이션이 상단에 고정됨
- **Mobile Menu Toggle**: 모바일에서 햄버거 메뉴로 네비게이션 제공
- **그래디언트 배경**: CSS 라디얼/리니어 그래디언트로 시각적 깊이 표현

## 🎨 디자인 특징

### 색상 팔레트
- **배경**: `#fdfbf7` (따뜻한 베이지)
- **강조색**: `#ff7a3c` (활기찬 오렌지)
- **텍스트**: `#222222` (다크 그레이)
- **보조 텍스트**: `#72748a` (뮤트 그레이)

### 타이포그래피
- **폰트**: Pretendard (한글/영문 최적화)
- **헤더**: 600-700 weight
- **바디**: 400-500 weight

### 라운드 처리
- **보더 레이디우스**: 18px로 부드러운 모서리

## 🚀 사용 방법

### 로컬에서 실행
1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/LeeBellHa/NewMusic_Portfolio.git
   ```

2. 프로젝트 폴더로 이동합니다:
   ```bash
   cd NewMusic_Portfolio
   ```

3. `index.html`을 웹 브라우저에서 엽니다:
   - 파일 탐색기에서 더블클릭하거나
   - 터미널에서 실행: `open index.html` (macOS) 또는 `start index.html` (Windows)

### 배포
- GitHub Pages, Vercel, Netlify 등 정적 사이트 호스팅 서비스에 배포 가능합니다
- 별도의 빌드 과정이 필요 없습니다 (순수 HTML/CSS/JavaScript)
